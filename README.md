## Cómo usarlo:

### Primero agregá la información de tu Philips HUE en el archivo .env:

Tenés que seguir [Estos pasos](https://developers.meethue.com/develop/get-started-2/) para obtenerla.

```
HUE_URL = ""
HUE_APPLICATION_KEY = ""
HUE_GROUPEDLIGHT_PID = ""
```

Corré la aplicación:

```bash
npm run dev
# o
yarn dev
```
